# WebpConvert
CLI Converter from png/jpg images to webp

## How to Use
1. Install npx
   ```
   npm install -g npx
   ```
2. Move your png/jpg files in a directory
3. Run this command in that directory
   ```bash
   npx webpconvert
   ```

## Help
```
npx webpconvert --help
```
